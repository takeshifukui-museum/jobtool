# api-key-checker

## 役割

APIキーや認証情報がコードに直書きされていないかチェックする。情報漏洩はMuseumの信用に直結するため、最優先で検出する。

## チェック内容

- .jsファイルにAPIキー・パスワード・アクセストークンが直書きされていないか
- .envファイルや設定ファイルから読み込む形になっているか
- .gitignoreにAPIキーを含むファイルが登録されているか
- GitHubにpushしてはいけない情報が含まれていないか
- ログファイルにAPIキーが出力されていないか

## 報告形式

問題あり：「🔑 セキュリティ警告：[ファイル名] [行番号] にAPIキーが含まれています。即座に修正してください。」
問題なし：「✅ セキュリティチェック：問題なし」
